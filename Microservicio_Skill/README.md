INFORMACIÓN GENERAL
===================
Este sistema implementa un ecosistema de microservicios con Spring Boot y Java 21, diseñado bajo un enfoque de arquitectura limpia. El núcleo es un Servidor Eureka que gestiona el descubrimiento de servicios, permitiendo que el microservicio de productos y el de clientes (generado con arquitectura avanzada) se registren y comuniquen de forma dinámica. El servicio de clientes destaca por utilizar capas de servicio, DTOs y manejo global de excepciones, garantizando que la lógica de negocio esté separada de la exposición de datos y permitiendo una escalabilidad superior a los métodos tradicionales.

Para ejecutar el proyecto, primero se debe iniciar el eureka-server en el puerto 8761 (http://localhost:8761/)para habilitar el panel de control. Posteriormente, se ejecutan los microservicios producto-service y cliente-service de manera independiente mediante el comando ./gradlew bootRun; una vez activos, ambos aparecerán registrados automáticamente en el dashboard de Eureka. Esto permite probar los endpoints REST, incluyendo funciones avanzadas como la paginación y validación de datos, directamente desde el navegador o herramientas como Postman.
